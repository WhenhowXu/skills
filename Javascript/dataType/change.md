- `Number(object)`
  > 转数字 可以转化成数字的转成数字，不能的转为 NaN

```js
Number(); // 0
Number(true); // 1
Number(false); // 0
Number(new Date()); // 1689575170388
Number(null); // 0
Number(""); // 0
Number("123a"); // NaN
Number(undefined); // NaN
Number({}); // NaN
```
