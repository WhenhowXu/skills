### DOM

> 文档对象模型（Document Object Model），是 W3C（万维网联盟）的标准，DOM 是中立于平台和语言的接口，它允许程序和脚本动态地访问和更新文档的内容、结构和样式

### BOM

> 浏览器对象模型（Browser Object Model），独立于内容而与浏览器窗口进行交互的对象，用于访问浏览器的功能，事实标准
